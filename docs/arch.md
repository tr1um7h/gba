## 1. 项目概述

GBA 是一个基于 Claude Agent SDK 构建的 AI 辅助功能开发工具。它通过结构化的工作流帮助开发者：

- **规划功能** - 与 AI 交互式讨论并生成设计规范
- **自动实现** - 按阶段执行实现计划
- **代码审查** - AI 驱动的代码审查和修复循环
- **PR 创建** - 自动生成 Pull Request

### 核心特性

```
┌─────────────────────────────────────────────────────────────────┐
│                        GBA 核心特性                              │
├─────────────────────────────────────────────────────────────────┤
│  • 交互式 TUI 规划界面                                           │
│  • 基于 Git Worktree 的隔离开发                                  │
│  • 可配置的任务模板系统                                          │
│  • 流式输出与实时进度显示                                        │
│  • 完整的状态持久化与恢复                                        │
│  • 自动代码审查与验证循环                                        │
│  • GitHub PR 集成                                                │
└─────────────────────────────────────────────────────────────────┘
```

## 3. Crate 详解

### 3.1 gba-cli (CLI 应用)

**位置**: `apps/gba-cli/`

```
apps/gba-cli/
├── Cargo.toml
└── src/
    ├── main.rs          # 入口点、信号处理、日志初始化
    ├── cli.rs           # Clap 命令行定义
    ├── config.rs        # .gba/config.yml 配置类型
    ├── state.rs         # Feature 状态管理
    ├── error.rs         # CLI 错误类型
    ├── utils.rs         # 工具函数 (路径、ID生成等)
    ├── commands/
    │   ├── mod.rs
    │   ├── init.rs      # gba init
    │   ├── plan.rs      # gba plan
    │   ├── run.rs       # gba run
    │   ├── list.rs      # gba list
    │   ├── status.rs    # gba status
    │   └── clean.rs     # gba clean
    └── tui/
        ├── mod.rs
        ├── app.rs       # Plan TUI 应用
        ├── run_app.rs   # Run TUI 应用
        └── widgets/     # TUI 组件
```

**职责**:
- 命令行参数解析
- 用户交互界面 (TUI)
- 状态管理与持久化
- Git 操作封装

### 3.2 gba-core (核心引擎)

**位置**: `crates/gba-core/`

```
crates/gba-core/
├── Cargo.toml
└── src/
    ├── lib.rs           # 公开 API
    ├── engine.rs        # 核心执行引擎
    ├── session.rs       # 多轮会话管理
    ├── task.rs          # 任务类型定义
    ├── config.rs        # Engine 和 Task 配置
    ├── event.rs         # EventHandler 特征
    └── error.rs         # 错误类型
```

**核心组件**:

```
┌─────────────────────────────────────────────────────────────────┐
│                         Engine                                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  pub struct Engine<'a> {                                        │
│      workdir: PathBuf,                                          │
│      prompts: PromptManager<'a>,                                │
│      base_options: Option<ClaudeAgentOptions>,                  │
│  }                                                              │
│                                                                 │
│  ┌───────────────────────────────────────────────────────────┐  │
│  │ 单次任务                                                   │  │
│  │  • run(task) -> TaskResult                                │  │
│  │  • run_stream(task, handler) -> TaskResult                │  │
│  └───────────────────────────────────────────────────────────┘  │
│                                                                 │
│  ┌───────────────────────────────────────────────────────────┐  │
│  │ 交互式会话                                                 │  │
│  │  • session(id) -> Session                                 │  │
│  │  • session_with_task(kind, ctx, id) -> Session           │  │
│  └───────────────────────────────────────────────────────────┘  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│                         Session                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  pub struct Session {                                           │
│      client: ClaudeClient,                                      │
│      session_id: String,                                        │
│      history: Vec<ConversationMessage>,                         │
│      stats: TaskStats,                                          │
│      connected: bool,                                           │
│  }                                                              │
│                                                                 │
│  方法:                                                          │
│  • connect() - 建立连接                                         │
│  • send(message) -> String - 同步发送                           │
│  • send_stream(msg, handler) -> String - 流式发送               │
│  • stats() -> &TaskStats - 获取统计                             │
│  • disconnect() - 断开连接                                      │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3.3 gba-pm (提示管理)

**位置**: `crates/gba-pm/`

```
crates/gba-pm/
├── Cargo.toml
└── src/
    ├── lib.rs
    ├── manager.rs       # PromptManager 实现
    └── error.rs         # 错误类型
```

**功能**:
- 加载 `.j2` / `.jinja` / `.jinja2` 模板
- 使用 MiniJinja 渲染模板
- 支持目录批量加载

```
PromptManager
    │
    ├── load_dir(path) ────────▶ 扫描 tasks/*/
    │                            加载所有模板文件
    │
    └── render(name, context) ──▶ 渲染模板
                                  返回 String
```
